## 💫 Animasyonlar

1. **fadeInUp**: Sayfa yüklenirken içeriklerin yukarıdan kayması
2. **pulse**: WhatsApp ikonunun nabız gibi atması
3. **bounce**: Harita ikonunun zıplaması
4. **gradientShift**: Arka plan gradient'inin değişmesi
5. **hover transforms**: Kartların yakınlaşması ve gölge değişimi

## 🎯 Responsive Breakpoints

- **Desktop**: > 900px
- **Tablet**: 768px - 900px
- **Mobile**: < 768px
